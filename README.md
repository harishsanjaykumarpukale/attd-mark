# qrcode

A flutter app to mark attendance BY SCANNING QR CODE

## Requirements
1. QR CODE must contain JSON data in following format
    {
     "doc_id": "<DOCUMENT ID>",
     "eid": "EVENT ID"
    } 
2. EVENT ID must match .(#TO BE DONE)

3. The User must be present in(near) TEMPLE(13 N,57 E). (#TO BE DONE)
